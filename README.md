# 🌐 Portfolio Website – Final Project

## 📌 Overview

This project is a personal portfolio website developed as the final version of previous assignments.
It demonstrates modern front-end development concepts including dynamic content, API integration, and user interaction.

The website showcases my projects, technical skills, and contact information in a clean, responsive, and interactive interface.

## 🌍 Live Demo

👉 [View Portfolio](https://fatimah-alshehab.github.io/202278660-fatimahalshehab-assignment4/)

---

## 🚀 Features

### 🎨 User Interface

* Responsive design for different screen sizes
* Light/Dark mode toggle with saved preference
* Smooth scrolling and section reveal animations

### 🧠 Advanced Functionality

* Project filtering by category
* Visitor timer tracking time spent on the site
* Dynamic greeting based on time of day
* Show More / Show Less feature for projects

### 🔗 API Integration

* Fetches repositories dynamically using GitHub API
* Displays repository name, description, and link
* Includes error handling for failed requests

### 📩 Form Validation

* Ensures all fields are filled
* Validates email format using regex
* Displays success and error messages

### ⚙️ Performance Optimization

* Lazy loading for images
* Optimized DOM manipulation
* Efficient event handling
